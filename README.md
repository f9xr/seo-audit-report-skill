# F9XR's SEO CodeBase Auditor SKILL

**What is SEO CodeBase Auditor?** An enterprise-grade AI-powered SEO auditing tool for static code repositories. Drop one file, run one prompt, get production-ready fixes across 24 SEO pillars in under 2 minutes.

## What does SEO CodeBase Auditor do?

SEO CodeBase Auditor analyzes your codebase for 24 critical SEO factors and provides actionable fix blocks. It's designed for developers, SEO professionals, and agencies who need fast, accurate technical SEO audits without manual checks.

## Quick Start: How to Use SEO CodeBase Auditor

1. Download [SKILL.md](SKILL.md) into your project root
2. Open your AI tool (Cursor, Copilot, Claude, etc.)
3. Run: `@SKILL.md Run a comprehensive full-stack SEO audit`
4. Receive `seo_audit_report.md` and `seo_audit_report.csv`

## Key Features of SEO CodeBase Auditor

- **24 SEO pillars** - from technical crawlability to YouTube SEO
- **Context-aware** - portfolio sites and e-commerce stores get different audits
- **Fix blocks** - every finding includes the exact code replacement
- **CSV export** - spreadsheet-ready issue tracker included
- **AI-powered** - works with Cursor, Copilot, Claude, and other AI coding assistants
- **Production-ready** - enterprise-grade audit results you can deploy immediately

## What SEO Factors Does It Audit?

SEO CodeBase Auditor covers 24 critical SEO pillars including:

- Technical SEO (crawlability, indexability, site speed)
- On-page SEO (meta tags, headings, content structure)
- Off-page SEO (backlinks, social signals)
- Local SEO (Google Business Profile, NAP consistency)
- E-commerce SEO (product pages, schema markup)
- YouTube SEO (video optimization, channel structure)
