# F9XR's SEO CodeBase Auditor SKILL

Enterprise-grade SEO auditing for static code repositories. Drop one file, run one prompt, get production-ready fixes across 24 pillars.

## Quick Start

1. Download [SKILL.md](SKILL.md) into your project root
2. Open your AI tool (Cursor, Copilot, Claude, etc.)
3. Run: `@SKILL.md Run a comprehensive full-stack SEO audit`
4. Receive `seo_audit_report.md` and `seo_audit_report.csv`

## Features

- **24 SEO pillars** - from technical crawlability to YouTube SEO
- **Context-aware** - portfolio sites and e-commerce stores get different audits
- **Fix blocks** - every finding includes the exact code replacement
- **CSV export** - spreadsheet-ready issue tracker included
